# Telegram Bot to Update User on Stake Crash game
# To run this bot:
Make an .env file and add 
```
TELEGRAM_API=YOUR_API_KEY
TELEGRAM_CHANNEL=@YOUR_CHANNEL_NAME
```
and simply run 
```
pip3 -r requirements.txt
python3 main.py
```
Bot usages
```
/under 1.02
/above 1.5
/equal 2.69
```
The bots send alert based on bots command set.
